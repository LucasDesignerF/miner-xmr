Dashboard Miner Control

O sistema definitivo para gerenciamento de mineração de criptomoedas.

Recursos Principais

Suporte a CPU e GPU

Otimização dinâmica para qualquer hardware, com suporte a mineração em VPS e contêineres Docker.

Monitoramento em Tempo Real

Acompanhe hashrate, uso de recursos, temperatura e status dos mineradores em tempo real.

Segurança Avançada

Criptografia de ponta a ponta, autenticação multifator (MFA) e proteção contra ataques DDoS.

Integração com Carteiras

Vinculação com Binance, Coinbase e outras carteiras populares para automação de pagamentos.

Tecnologias Utilizadas

Python & Flask

Back-end robusto e escalável para gerenciamento de dados e APIs.

MongoDB

Banco de dados NoSQL para armazenamento dinâmico e eficiente.

Docker

Contêinerização para fácil implantação e escalabilidade.

OAuth2 & MFA

Autenticação segura via GitHub, Google e Microsoft com MFA.
